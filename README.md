# Excel-Croatian_salaries_analysis
An interactive Excel Data Dashboard that analyses Gross and Net salaries in Croatia over the span of 5 and 8 years (for critical KPI indicators).

# Key Features
**Interactive Map Visualization**: visualizes salary distribution across Croatian counties using Excel’s native map chart.

**Gender Pictographs**: highlights salary discrepancies between men and women through intuitive icon‑based visuals.

**Sector Comparison Graph**: compares salary levels across major economic sectors.

**KPI indicators**: critical KPI indicators that represent salary variations among 4 main categories.    

# Workflow
1. Data cleaning and transformation
  - Raw data sourced from the Croatian Bureau of Statistics ([DZS](https://www.https://podaci.dzs.hr/hr/statistika-u-nizu/example.com)).  
  - Imported into Power Query for cleaning, normalization, and restructuring.  
  - Applied M language transformations (column type adjustments, filtering, deduplication, and formatting).  
    
2. Power Pivot and DAX
  - Cleaned tables loaded into Power Pivot for data modeling.
  - Created DAX measures for KPI calculations.

3. Visuals
  - Designed visuals to support dynamic exploaration of trends and added slicers.

# How to use the Dashboard
**Enable Content**: upon opening the .xlsx file for the first time, click **"Enable Content"** on the top yellow security banner to allow the Power Pivot connection to load.

**Filter the Timline**: use the **Year** and **Month** **Slicers** to narrow down data.

**Explore Regions**: interact with the slicers to watch the Map Chart redraw itself.
